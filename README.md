Amazon Website Clone 

This is a simple **Amazon-inspired website** built using **JavaScript, HTML, and CSS**.  
The project focuses on recreating the basic layout and functionality of an e-commerce website for learning and practice purposes.

Features
- Homepage with product listings
- Add to cart functionality
- Responsive design
- Basic UI similar to Amazon

 Tech Stack
- HTML
- CSS
- JavaScript

 How to Run
1. Clone the repository  
2. Open `index.html` in your browser
